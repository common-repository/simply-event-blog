== Description ==

Adds events feature for posts. It's compatible with these themes:
* GeneratePress.


Others Closemarketing Premium Plugins:
- [Closemarketing.NET](https://closemarketing.net/productos-online/category/plugin-wordpress/?utm_source=WordPress.org)

== Installation ==

1. Upload `simply-event-blog.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
